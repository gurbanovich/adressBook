The task:

Implement a version of the address book that will store information about people, each person can have 1 or more emails, 1 or more phone numbers, 1 address, 1 comment field, as well as fields such as last name, first name, place of work.

It is necessary to create and edit information, view individual information, view all records in a table in abbreviated form (only basic information - name, 1 email, 1 phone).

No need to attach any authorization.

Technologies for implementation - the server part is preferably JEE, using the Spring stack (spring-boot), the client part is implemented using the Anguar 2+ framework, or AngularJS 1.x.
